02

Easy Order (Best → Worst)

O(1) → O(log n) → O(√n) → O(n) → O(n log n) → O(n²) → O(n³) → O(2ⁿ) → O(n!)
03

Interview Tip

● Chase these

  • Excellent: O(1), O(log n)
  • Good: O(n), O(n log n)

● Watch out

  • Acceptable (small n): O(n²)
  • Avoid if n large: O(n³), O(2ⁿ), O(n!)
